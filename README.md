# RU

# Последовательные одномерные контейнеры

Данная работа посвящена изучению алгоритмов формирования и обработки одномерных массивов и последовательных контейнеров. В рамках лабораторной работы реализованы численные методы решения нелинейных уравнений, алгоритмы обработки массивов и работа с генераторами случайных чисел.

---

## Цели работы

- Изучение алгоритмов формирования и обработки одномерных массивов
- Освоение методов решения нелинейных уравнений
- Работа с генераторами случайных чисел
- Программирование и отладка вычислительных алгоритмов

---

## Требования

- Компилятор C++ (g++, clang или MSVC)
- Среда разработки (Visual Studio, CLion, VS Code и т.д.)

---

## Как использовать

1. Открыть соответствующий файл задания.
2. Скомпилировать программу выбранного языка.
3. Запустить программу.
4. Ввести необходимые параметры.
5. Получить результат вычислений в консоли.

5. Следовать инструкциям в консоли (если предусмотрен ввод данных).

---

## Задания

### Задание №1 — Методы решения нелинейных уравнений

- Отделение корней уравнения
- Уточнение корней с точностью до ε = 10⁻⁴
- Реализация методов:
  - Метод половинного деления
  - Метод Ньютона
  - Метод простых итераций
  - Метод хорд

### Задание №2 — Вычисления в одномерных массивах

- Формирование массива
- Обработка элементов массива
- Выполнение вычислительных операций

### Задание №3 — Работа с генераторами случайных чисел

- Использование встроенных генераторов
- Анализ распределения значений

### Задание №4 — Дилемма заключенного

- Реализация стратегий поведения
- Моделирование игрового процесса
- Анализ результатов

### Задание №5 — Генерация псевдослучайных чисел

- Реализация алгоритма генерации
- Проверка корректности работы генератора
- Анализ полученной последовательности

---

## Используемые технологии

- C++
- STL
- Численные методы

---

## Вывод

В ходе выполнения лабораторной работы были изучены алгоритмы обработки одномерных массивов, методы решения нелинейных уравнений и принципы генерации случайных чисел. Работа позволила закрепить навыки разработки и анализа вычислительных алгоритмов.
